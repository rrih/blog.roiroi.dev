Git には、あるブランチの変更を別のブランチに統合するための方法が大きく分けて二つあります。 merge と rebase です。

まず git merge について

git merge は conflict 解消する時とかにこんな感じで使ってる 

$ git branch # master
$ git pull
$ git checkout working_branch # 作業ブランチへ移動
$ git merge origin master # 最新の master を作業ブランチへ merge
$ git status
